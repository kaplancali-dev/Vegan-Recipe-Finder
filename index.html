📲 Sync Between Devices
Laptop → iPhone:
1. On your laptop, tap Send My Data below, then Copy Code
2. Send the code to yourself (paste it into Notes, text it to yourself, or email it)
3. On your iPhone, open that note/text and copy the code
4. Open HARVEST → Favorites → tap Receive Data → paste the code → tap Apply
iPhone → Laptop:
1. On your iPhone, tap Send My Data below, then Copy Code
2. Send the code to yourself (paste it into Notes, text it to yourself, or email it)
3. On your laptop, open that note/text and copy the code
4. Open HARVEST → Favorites → tap Receive Data → paste the code → tap Apply
